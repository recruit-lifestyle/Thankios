# Thankios
Acknowlegements Settings.bundle generator for CocoaPods and Carthage.

## Installation
```bash
$ brew tap uny/apple
$ brew install thankios
```

## Usage
```bash
# On your project root:
$ thankios <destination path>

# Example
$ thankios ~/SampleProject/SampleProject/Settings.bundle/
```

## Requirements
* Xcode 7.3+
* OSX 10.11+
